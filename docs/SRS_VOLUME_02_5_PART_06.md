# AI Documentary Studio - SRS Volume 02.5 Part 06 - Visual Rhythm Engine

**Version:** 1.0 **Status:** Critical **Priority:** Highest

## Objective
Control pacing of the documentary based on viewer attention, not editing speed. Continuously adjusts visual rhythm so audience stays engaged without feeling overwhelmed or bored.

## Position
Sync Engine → Motion Timeline Engine → Visual Rhythm Engine → Renderer

## Rhythm Unit
Each block: Visual Tempo, Attention Score, Information Density, Energy Level, Cognitive Load.

## Attention Curve
Continuous curve: Hook↑ → Curiosity↑ → Explanation↓ → Reflection↓ → Reveal↑ → Ending↑. Timeline follows this curve.

## Information Density
Low/Medium/High/Extreme. High density = longer holds.

## Cognitive Load Engine
Factors: abstract concepts, fast narration, new terminology, complex visuals. High load = reduce editing speed, reduce motion, longer holds.

## Visual Fatigue Detector
Monitor rapid cuts, too much motion, too many effects, frequent camera changes. If fatigue rises → slow timeline.

## Hold Rhythm Engine
Optimal hold = f(concept difficulty, emotion, narration speed, image complexity, viewer processing time).

## Cut Rhythm Engine
Cuts when meaning changed, never when time expired.

## Motion Rhythm
Hold → Push → Hold → Pan → Hold. Constant movement prohibited.

## Visual Breathing
Intentional stillness. Image → Slow Push → Pause → Narration → Tiny Drift → Pause. Stillness improves comprehension.

## Curiosity/Reveal/Emotional Rhythm
Question → delay image change. Reveal → prepare → pause → transition → new visual → hold → explanation. Each emotion modifies rhythm (reflection=slow, fear=tense, wonder=floating, urgency=faster, hope=expansive).

## Visual Tempo Score
Slow / Medium / Fast / Dynamic. Documentary = mostly Slow or Medium.

## Viewer Attention AI
Predict attention, detect drop, predict boredom, recommend camera movement, longer holds, stronger emphasis.

## Cognitive Pacing Engine
Pace according to human understanding: information received → processed → understood. If understanding lags → extend hold, reduce motion, delay transition.

## Rhythm Signature System
Each channel stores Rhythm Signature. Documentary=calm slow, Educational=medium with emphasis, Motivational=rising tempo, Storytelling=alternating, News=fast with minimal holds.
