# AI Documentary Studio - SRS Volume 02 Part 07 - Visual DNA System

**Version:** 1.0  
**Status:** Critical  
**Priority:** Highest  

## 1. Objective

Define the permanent visual identity of a channel. Controls how every scene looks regardless of topic.

## 2. Design Philosophy

Every channel has its own visual identity. Preserve it automatically. Never manually redefine style per project.

## 3. Hierarchy

Channel DNA → Visual DNA → Project Visual Profile → Visual Storyboard → Prompt Engine → Image Library → Video

## 4. Components

Art Style, Character DNA, Environment DNA, Camera Rules, Lighting Rules, Composition Rules, Color Rules, Effects Rules, Consistency Rules, Visual Metaphor Rules

## 5. Art Style

Art Technique, Rendering Style, Illustration Style, Outline Style, Texture, Shadow, Depth, Background, Quality Target

## 6. Character DNA

ID, Body Structure, Head Shape, Pose, Expression, Gestures, Walking, Emotion, Scale, Silhouette. Never redefine in prompt.

## 7. Environment DNA

Type, Object Density, Background Complexity, Perspective, Indoor/Outdoor Rules, Depth, Negative Space

## 8. Color Language

Primary, Secondary, Accent, Danger, Success, Neutral, Night Palette, Day Palette. Consistent across videos.

## 9. Lighting DNA

Direction, Intensity, Mood, Temperature, Shadow Density, Highlight, Backlight, Ambient

## 10. Camera DNA

Default, Wide, Medium, Close-up, Extreme Close-up, Top, Side, Perspective Rules

## 11. Composition DNA

Rule of Thirds, Centered, Leading Lines, Negative Space, Balance, Eye Direction, Reading Flow, Object Placement

## 12. Visual Metaphor Library

Mandatory. Memory→Archive, Identity→Mirror, Fear→Shadow, Hope→Sunrise, Freedom→Open Sky, Control→Puppet Strings, Time→Hourglass, Stress→Cracks, Focus→Spotlight, Habits→Circular Loop, Dopamine→Glowing Pulse, Silence→Empty Room

## 13. Visual Symbol Rules

Meaning, Visual Style, Reuse Rules, Animation Preference, Camera Preference, Emotion Mapping. Symbols must stay recognizable.

## 14. Visual Priority System

Critical, High, Medium, Low. Critical = new image, strong composition, high detail, longer screen time. Low = reuse existing.

## 15. Image Complexity Profile

Simple, Medium, Detailed, Complex. Match complexity to scene purpose.

## 16. Consistency Engine

Verify character, environment, lighting, camera, color, art style, perspective, object scale before prompt approval.

## 17. Visual Rhythm Profile

Define how visuals change over time (Wide→Medium→Close-up→Word→Wide→Symbolic). Avoid repetitive patterns.

## 18. Image Reuse Profile

Hold during explanation, slow zoom for reflection, pan for continuation, new image only on meaning change.

## 19. Project Visual Overrides

Project DNA may override color mood, lighting mood, camera preference, symbol set, density. Core identity unchanged.

## 20. Visual Validation

Before Image Library entry: character, art style, camera, lighting, symbol, composition, color harmony.

## 21. Visual Grammar Engine

Define image-to-image relationships: continue, zoom, pan, word insert, return, new scene. Enforce visual storytelling grammar.

## 22. Zenn-style Built-in Profile

Minimal flat vector, stick figures, black line weight, pastel accent colors, geometric environments, medium/close framing, soft ambient lighting, strong negative space, symbolic metaphors.
