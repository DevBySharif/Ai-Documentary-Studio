# VOLUME 02.6 — PART 04: CHARACTER & STYLE CONSISTENCY ENGINE

**Version:** 1.0
**Status:** Critical
**Priority:** Highest

## 1. OBJECTIVE

The Character & Style Consistency Engine ensures every generated visual remains faithful to the active Channel DNA. The viewer should immediately recognize the channel's artistic signature.

## 2. STYLE DNA

Every channel has a permanent Style DNA: Art Style, Line Style, Lighting, Contrast, Texture, Perspective, Mood. The generator must never violate the Style DNA.

## 3. CHARACTER DNA

Every recurring character receives a Character DNA: Character ID, Body Shape, Head Ratio, Limb Length, Pose Rules, Emotion Rules, Silhouette, Accessories, Identity. Same character must appear identical across all projects.

## 4. STYLE & CHARACTER LOCK

Locked attributes: Art Style, Line Weight, Background, Lighting, Composition, Camera, Mood, Face, Eyes, Head Shape, Stick Proportions, Movement Style, Silhouette. These cannot be overridden by the prompt.

## 5. COLOR LANGUAGE

Each channel defines: Base, Accent Colors, Highlights, Background. Future prompts inherit this automatically.

## 6. SYMBOL LIBRARY

Permanent symbol-to-meaning mappings (Mirror→Identity, Bird→Freedom, Galaxy→Infinity, Brain→Knowledge, Door→Opportunity). Never assign conflicting meanings.

## 7. PROMPT SANITIZER

Before sending to Google Flow, remove: Conflicting Styles, Anime/3D/Photorealistic References, Conflicting Lighting, Conflicting Character Descriptions. Only Channel DNA survives.

## 8. STYLE & CHARACTER VALIDATOR

After generation, compare against reference. Failed images are rejected for regeneration. Character checks: Silhouette, Proportions, Identity, Pose, Expression, Accessories.

## 9. STYLE REFERENCE BOARD

Stores: Hero Character, Background, Lighting, Composition, Camera, Symbol, Gold Standard references. Every new generation is compared against this board.

## 10. PROMPT DNA COMPILER

Before reaching Google Flow, compile: Scene Intent → Visual Intent → Style DNA → Character DNA → Composition Rules → Lighting Rules → Negative Rules → Final Prompt. Guarantees structured prompt format.

## 11. STYLE DRIFT DETECTOR

Monitor for drift: Line thickness, Proportions, Perspective, Lighting, Background, Color Palette. If drift exceeds threshold, regenerate only affected assets.
